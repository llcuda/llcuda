## ‚ùì FAQ

### Why was PyPI removed?

llcuda is now distributed exclusively through GitHub for:
- Faster updates
- Direct source access
- Simplified distribution
- Better version control
- No external platform dependencies

### Can I still use old PyPI versions?

No, the PyPI project has been deleted. All users must migrate to GitHub installation.

### How do I upgrade from PyPI version?

```bash
# Uninstall old PyPI version
pip uninstall llcuda -y

# Install from GitHub
pip install git+https://github.com/waqasm86/llcuda.git
```
