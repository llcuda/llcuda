# llcuda v2.0.x Archive

This directory contains archived documentation and scripts from llcuda v2.0.x releases (v2.0.1, v2.0.2, v2.0.6).

## Contents

### Root Files
- `BUNDLED_BINARIES_GUIDE.md` - Guide for bundled CUDA 12 binaries (v2.0.2)

### docs/
- `RELEASE_NOTES_v2.0.1.md` - Release notes for v2.0.1
- `GITHUB_RELEASE_DESCRIPTION_v2.0.1.md` - GitHub release description for v2.0.1
- `release-description-short.md` - Short release description (v2.0.1)
- `UPLOAD_INSTRUCTIONS.md` - PyPI upload instructions (v2.0.1)
- `UPLOAD_COMPLETE.md` - Upload completion notes (v2.0.1)
- `updated-release-notes.md` - Updated release notes (v2.0.1)

### scripts/
- `prepare_github_release_v2.0.2.sh` - GitHub release preparation script for v2.0.2
